# [BArista RObot: BARO]

## 📑 프로젝트 개요

- **BARO**
    
    SCARA 구조 기반 모듈형 바리스타 로봇 시스템
    
- **제작 목적**
    
    시중 바리스타 로봇 시스템에 비해 저렴한 비용 및 모듈형 구조를 통한 유지보수 효율성을 높여, 예비 창업자들의 진입장벽을 낮추고 커피 산업 내 자동화 기술의 대중화를 목표로 함.
